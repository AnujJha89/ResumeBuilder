# ResumeBuilder

A professional, feature-rich ASP.NET Core Web application designed to help users build, customize, and download high-quality resumes. Built with clean architecture principles and modern styling refinements.

## 🚀 Features

- **Interactive Dashboard**: View, create, edit, and delete your resumes easily from a unified dashboard.
- **Dynamic Editors**: 
  - **Personal Information**: Edit contact details and upload profile pictures.
  - **Work Experience**: Track employment history, job responsibilities, and timelines.
  - **Projects**: Highlight your projects, technologies used, and GitHub links.
  - **Skills & Education**: Document skills with proficiency categories and detailed education history.
- **Custom Templates & Layout Controls**: Adjust styling, custom page margins, font spacing, and settings.
- **High-Quality PDF Generation**: Download print-ready PDF copies of your resumes instantly.

## 🛠️ Tech Stack

- **Core**: ASP.NET Core MVC (C#)
- **Architecture**: Clean Architecture pattern (Core, Infrastructure, Web, and Tests layers)
- **Database**: Entity Framework Core
- **Authentication**: ASP.NET Core Identity
- **Front-end**: HTML5, CSS3, JavaScript, Bootstrap
- **PDF Rendering**: wkhtmltopdf / custom PDF Service
